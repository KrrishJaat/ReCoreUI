
## About ReCoreUI

ReCoreUI is A refined OneUI port framework built for stability, modularity, and performance.  
It automatically adapts and patches supported Samsung firmware builds while maintaining stability, performance, and feature richness.

Simply match the correct branch to your OneUI version, and ReCoreUI handles the rest.


### Project Aim

To provide a modular, device-adaptive OneUI port framework that can patch supported Samsung firmware builds and generate a flashable ROM package with minimal manual intervention.
